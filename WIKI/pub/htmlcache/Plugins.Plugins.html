All third party plugins

TabSRMM supports or is supported by a number of other plugins, most of them messaging related. Their use is optional though and TabSRMM does not require any other plugin with the exception of most core plugins to work but can benefit from features provided by said plugins.

This list describes some of the important and commonly used plugins.

  • IEView plugin (HTML based message log viewer for Miranda IM).
  • Popups for notifications (plugin(s) to display event notifications in MIranda IM).
  • Smileyadd (plugin for graphical smileys or emoticons).
  • Custom profile folders (plugin to manage storage folder locations in Miranda)
  • Meta contacts? (a protocol plugin to implement meta entities for contact management).
  • avatar service (core plugin for contact picture management).
  • Miranda image services (core plugin for low level image handling, based on the freeimage library.
  • Sound volume plugin (old plugin to control the volume of Miranda sound effects, using a convenient contact list frame)
  • Database editor plugin


All these plugins are developed by various members of the Miranda community and are not my work with the exception of some core plugins (AVS and Image services).
